# ToDo

## Описание
**ToDo** — это простое приложение на Kotlin, которое демонстрирует работу с базой данных SQLite. Приложение позволяет создавать, читать, обновлять и удалять записи в таблице пользователей.

Это отличный стартовый проект для тех, кто хочет изучить основы работы с базами данных SQLite в сочетании с Kotlin.

## Требования
Для запуска и использования приложения вам понадобятся следующие инструменты:
- **JDK 11+** или **Android SDK** (если вы используете Android Studio)
- **Kotlin** (версия 1.5+)
- **SQLite JDBC Driver** (если вы работаете вне Android)

Если вы используете Android Studio, убедитесь, что у вас установлены необходимые компоненты для разработки Android-приложений.
